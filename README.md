## About The Project

<br />

다양한 주제와 방식으로 토론하고 또한 직접 토론주제를 등록하고 사용자들과 의견을 나누는 하나의 토론 플랫폼입니다.

주요 기능:

- 로그인/로그아웃, 회원가입, 아이디 찾기 기능 제공
- 카테고리와 토론 분류에 맞게 여러 토론 게시물을 검색해 원하는 토론에 참여 할 수 있습니다.
- 로그인 한 이후 토론 게시물에 의견을 등록하고 등록된 의견들에 댓글 및 답글을 남겨 소통할 수 있습니다.
- 주제 추천 게시물을 등록해 원하는 주제를 등록할 수 있습니다.
- 어드민 페이지에서 토론게시물을 등록할 수 있습니다.
